Суперзадание

  1. Создайте пустой список для хранения покупок.
  2. С помощью метода append, добавьте три продукта в список.
  3. Добавьте еще два продукта разом, используя метод extend.
  4. Вставьте один продукт на вторую позицию.
  5. Удалите последний продукт из списка и сохраните его в переменной.
  6. Найдите индекс одного из продуктов в списке, и выведите его.
  7. Подсчитайте, сколько раз определенный продукт встречается в списке.
  8. Отсортируйте список в алфавитном порядке и выведите результат.
  9. Измените порядок элементов списка на обратный
  10. Создайте копию списка, добавьте 3 элемента и выведите обе версии.

Ответ


# 1. Создаем пустой список и добавляем продукты
shopping_list = []
shopping_list.append("milk")
shopping_list.append("bread")
shopping_list.append("butter")

# 2. Добавляем еще два продукта
shopping_list.extend(["eggs", "cheese"])

# 3. Вставляем продукт на вторую позицию
shopping_list.insert(1, "apple")

# 4. Удаляем последний продукт
last_item = shopping_list.pop()

# 5. Находим индекс определенного продукта
index_of_bread = shopping_list.index("bread")

# 6. Считаем количество повторений определенного продукта
count_of_apple = shopping_list.count("apple")

# 7. Сортируем список
shopping_list.sort()

# 8. Меняем порядок на обратный
shopping_list.reverse()

# 9. Создаем копию списка
copied_list = shopping_list.copy()
copied_list.extend(["orange", "apple", "kiwi"])

# Выводим результаты
print("Изначальный список:", shopping_list)
print("Удаленный продукт:", last_item)
print("Индекс 'bread':", index_of_bread)
print("Количество 'apple':", count_of_apple)
print("Копия списка:", copied_list)

Мега-супер-дупер-задание

  1. Создайте список, в котором будет храниться названия товаров в магазине.
  2. Создайте словарь, в котором для каждого товара будет указана его цена.
  3. Добавьте новые товары в список с помощью метода append.
  4. Добавьте новые цены для этих товаров в словарь с помощью метода update.
  5. Используйте метод insert, чтобы вставить новый товар в начало списка.
  6. Используйте метод remove, чтобы удалить один из товаров из списка.
  7. Найдите цену товара, используя метод get и передав название товара в качестве ключа.
  8. Подсчитайте, сколько товаров с определённой ценой есть в магазине, используя метод count для списка и методы словаря.
  9. Выведите на экран список всех товаров, их цены и общее количество товаров в магазине.

Ответ


# Шаг 1. Создаем список с названиями товаров
products = ["laptop", "phone", "headphones"]

# Шаг 2. Создаем словарь с ценами товаров
prices = {"laptop": 1000, "phone": 500, "headphones": 150}

# Шаг 3. Добавляем новые товары в список
products.append("tablet")
products.append("smartwatch")

# Шаг 4. Добавляем цены для новых товаров
prices.update({"tablet": 300, "smartwatch": 250})

# Шаг 5. Вставляем товар на начало списка
products.insert(0, "keyboard")

# Шаг 6. Удаляем один товар из списка
products.remove("headphones")

# Шаг 7. Находим цену товара
laptop_price = prices.get("laptop")
print(f"Цена товара 'laptop': {laptop_price}")  # 1000

# Шаг 8. Подсчитываем количество товаров с определенной ценой (например, цена 300)
count_300_price = list(prices.values()).count(300)
print(f"Количество товаров с ценой 300: {count_300_price}")  # 1

# Шаг 9. Выводим список товаров, их цены и общее количество
print("\nТовары в магазине:")
for product in products:
    print(f"{product}: ${prices.get(product)}")

print(f"\nОбщее количество товаров: {len(products)}")
