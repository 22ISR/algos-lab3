Метод extend


numbers = [1, 2, 3]  
numbers.extend([4, 5])  
print(numbers)  # [1, 2, 3, 4, 5]

Задание

Создайте список с названиями трех стран. Добавьте к нему еще два названия стран с помощью метода extend

Ответ


countries = ["USA", "Canada", "Mexico"]
countries.extend(["Brazil", "Argentina"])
print(countries)  # ["USA", "Canada", "Mexico", "Brazil", "Argentina"]

Метод pop


tasks = ["math", "science", "english"]  
last_task = tasks.pop()  
print(tasks)  # ["math", "science"]  
print(last_task)  # "english"

Задание

Создайте список из трех любимых блюд. Удалите последнее блюдо из списка и выведите его на экран.

Ответ


dishes = ["pizza", "sushi", "pasta"]
last_dish = dishes.pop()
print(dishes)  # ["pizza", "sushi"]
print(last_dish)  # "pasta"

Метод copy


original = [1, 2, 3]  
copied = original.copy()  
copied.append(4)  
print(original)  # [1, 2, 3]  
print(copied)  # [1, 2, 3, 4]

Задание

Создайте список из пяти элементов. Сделайте его копию с помощью метода copy и добавьте в копию новый элемент. Выведите оба списка.

Ответ


original = [1, 2, 3, 4, 5]
copied = original.copy()
copied.append(6)
print(original)  # [1, 2, 3, 4, 5]
print(copied)    # [1, 2, 3, 4, 5, 6]
